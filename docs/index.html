Why EasySQL?

专为企业复杂数据库场景设计的智能 SQL 生成引擎

DDD 业务上下文

深度理解业务领域，自动识别订单、库存、客户等核心概念，让 SQL 生成符合业务语义而非表面字段匹配。

知识图谱驱动

Neo4j 构建表关系图谱，精准捕获外键、索引、约束，确保多表 JOIN 路径最优且语义正确。

Few-Shot 智能学习

内置示例学习机制，通过少量标注样本快速适配特定业务场景，无需大规模训练数据。

越用越聪明

持续学习用户反馈和纠正，自动积累业务知识，查询准确率随使用时间稳步提升。

组合式架构

模块化设计，检索、生成、验证、修复各环节可独立配置，灵活组合满足不同场景需求。

语义向量检索

Milvus 驱动的深度语义理解，「本月销量」自动关联 order_detail，告别简单关键词匹配。

自愈式执行

SQL 执行异常时自动诊断错误根因，智能修复并重试，显著提升端到端成功率。

全栈数据库支持

MySQL、PostgreSQL、Oracle、SQL Server 一套方案通吃，统一接口屏蔽方言差异。

全链路可观测

LangFuse 集成，Token 消耗、推理耗时、检索召回一目了然，便于调优与成本控制。
